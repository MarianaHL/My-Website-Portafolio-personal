Proyectos recientes

FILM-MAR

FILM-MAR es una aplicación desarrollada en Android Studio con conexión a la plataforma de Firebase. En esta app puedes encontrar un catálogo de diferentes géneros cinematográficos y cada uno de estos cuenta con una lista de películas y al seleccionar una de estas se mostrará el tráiler y una pequeña información acerca de la película.

BlogApp

BlogApp es una app móvil desarrollada en Flutter con conexión a Firebase. Esta app te permite seleccionar una foto de tu galería de imágenes y agregarle una pequeña descripción que se mostrará al pie de la imagen. Estas se suben a la base de datos de Firebase para después poder mostrarlas en la aplicación.
